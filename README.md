# 在线答题系统

一个基于Vue.js和WebSocket的实时在线答题系统，支持多种题型和实时互动。

## 功能特点

- 🎯 **多种题型支持**：单选题、多选题、选词填空、开放填空
- 🔄 **实时同步**：WebSocket实现实时答题和状态同步
- 👥 **用户管理**：用户登录、姓名管理、完成率统计
- 📊 **管理面板**：题目管理、答案显示/隐藏、用户监控
- 🎨 **答案反馈**：答案公布时的颜色高亮显示
- 📱 **响应式设计**：支持桌面和移动设备

## 技术栈

- **前端**：Vue 3 + Element Plus + Vite
- **后端**：Node.js + WebSocket
- **数据存储**：内存存储 + localStorage

## 项目结构

```
online-quiz-vue/
├── src/
│   ├── components/          # Vue组件
│   ├── pages/              # 页面组件
│   │   ├── Login.vue       # 登录页面
│   │   ├── Quiz.vue        # 答题页面
│   │   └── Admin.vue       # 管理员页面
│   ├── router.js           # 路由配置

```

## 快速开始

### 1. 安装依赖

```bash
cd online-quiz-vue
npm install
```

### 2. 启动服务器

```bash
node server.cjs
```

### 3. 启动前端开发服务器

```bash
npm run dev
```

### 4. 访问应用

- 前端地址：http://localhost:3000
- 学生口令：65472
- 管理员口令：14725

## 使用说明

### 学生端
1. 输入姓名和口令登录
2. 等待管理员开放题目
3. 实时答题，答案自动保存
4. 查看公布的参考答案

### 管理员端
1. 使用管理员口令登录
2. 查看用户列表和完成率
3. 按顺序开放题目
4. 显示/隐藏答案
5. 随机抽人功能

## 题目类型

### 1. 单选题
- 四个选项中选择一个正确答案
- 答案显示时正确选项显示绿色，错误选项显示红色

### 2. 多选题
- 四个选项中选择多个正确答案
- 支持无序匹配

### 3. 选词填空
- 从给定词汇中选择填入空白处
- 输入框颜色反馈正确性

### 4. 开放填空
- 自由输入文本答案
- 支持部分匹配

## 开发说明

### 修改题库
编辑 `quizData.cjs` 文件来修改题目内容。

### 端口配置
- 前端端口：在 `vite.config.js` 中修改
- 后端端口：在 `server.cjs` 中修改（默认9527）
